# Nginx Docker

Example to initialize Nginx Docker

## Installation

### Start service

```bash
make start
```

### Stop service

```bash
make stop
```

## Configuration

- `NGINX_VERSION`: Nginx Docker version
- `NGINX_CONFIG_HOST`: Host path to store Nginx config at host
- `NGINX_HOST_PORT`: Nginx published host port
